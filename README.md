# Batch-helper
Batch helper for fast .bat file creation.

by Ddomer
https://github.com/UAPROGRAMER

made with python

fuction:
-auto convert file to .bat when saving
-panel with most used batch func

#HOTKEYS
Alt+s - create "start "
Alt+w - create loop
Alt+d - create "goto "
Control+s - save file
Control+Alt+s - save file as
Control+e - open file
Control+q - new file
